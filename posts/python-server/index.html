Python HTTP Server

February 16, 2023

Ever wonder about how to share a file in your home network easily? Or how to test the blog you just created on you local machine? All without the burden of downloading and configuring Nginx?

Wonder no more! Python is here to rescue you!

Python is shipped with a built-in library called http.server, and you can use it directly from the shell.

Go to directory with the files you want to share and execute the command below:

python3 -m http.server

Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...

As the output says, just write the URL http://localhost:8000 on your browser and you can see all the files on that directory. In the example above, the server is bound to 0.0.0.0, it means you can access it through any network interface, be it localhost or your wifi card.

You can also bind the server to a different port or IP

python3 -m http.server --bind 127.0.0.1 3000

Serving HTTP on 127.0.0.1 port 3000 (http://127.0.0.1:3000/) ...

Needless to say, don’t use it on production and don’t use it in a network you don’t trust!
